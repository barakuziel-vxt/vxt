## âš¡ How It Works

All colors are controlled through CSS variables in `src/index.css`:

```css
:root {
  --light-bg: #1a1a1a;           /* Main background */
  --white-bg: #2d2d2d;           /* Cards/sidebar */
  --text-color: #e0e0e0;         /* Bright text */
  --primary-color: #667eea;      /* Accent color */
  /* ...and more... */
}
```

**Benefit**: Change one variable = entire dashboard updates!
